# PowerBi_dashboard_project

## Project Overview
This repository contains a **Sales Dashboard built using Microsoft Power BI**.  
The project is based on the **SuperStore Sales Dataset** and focuses on analyzing sales, profit, customer segments, and regional performance.

The main purpose of this project is to practice **real-world Power BI concepts** and present insights in a clear and interactive format.

---

## Dataset Details
- **Dataset:** SuperStore Sales Dataset
- **File Name:** `SuperStore_Sales_Dataset.csv`

**Key Columns:**
- Order Date  
- Sales  
- Profit  
- Quantity  
- Category & Sub-Category  
- Customer Segment  
- Region & State  

This dataset is widely used for **sales and retail analysis practice**.

---

## Tools Used
- Microsoft Power BI Desktop
- Power Query Editor
- DAX (Data Analysis Expressions)
- CSV file as data source

---

## Project Workflow

### Step 1: Data Import
- Imported the SuperStore sales CSV file into Power BI
- Checked column names and data types

### Step 2: Data Cleaning
- Removed unnecessary columns
- Renamed columns for better understanding
- Checked for missing or incorrect values using Power Query

### Step 3: Data Modeling
- Used a single table model
- Enabled date hierarchy for time-based analysis

### Step 4: DAX Measures
Created basic measures such as:
- Total Sales
- Total Profit
- Total Quantity
- Profit Margin
- Sales over time

### Step 5: Dashboard Design
- KPI cards for Sales, Profit, and Quantity
- Line chart for sales trend analysis
- Bar charts for category and sub-category comparison
- Region-wise visualization
- Slicers for date, region, category, and segment

3. Update dataset path if required
4. Click **Refresh**
5. Explore the dashboard using slicers and visuals
